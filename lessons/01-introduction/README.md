## Что такое Go?
Go (или Golang) - это открытый язык программирования, разработанный в Google. Основные особенности:
- Простой и понятный синтаксис
- Встроенная поддержка конкурентности
- Быстрая компиляция
- Эффективный сборщик мусора
- Статическая типизация
- Кросс-платформенность

## Установка и настройка

### 1. Установка Go
- Скачайте установщик с [официального сайта](https://golang.org/dl/)
- Следуйте инструкциям установки для вашей операционной системы
- Проверьте установку командой:
```bash
go version
```

### 2. Настройка рабочего пространства
Go использует определенную структуру рабочего пространства. Создайте директорию для ваших Go проектов:
```bash
mkdir ~/go
cd ~/go
mkdir src bin pkg
```

### 3. Настройка IDE
Рекомендуемые IDE:
- VSCode с расширением Go
- GoLand
- Vim/Neovim с Go плагинами

## Первая программа

Создадим классическую программу "Hello, World!":

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```

Давайте разберем каждую строку:
- `package main` - объявление пакета (main - точка входа в программу)
- `import "fmt"` - импорт пакета для форматированного вывода
- `func main()` - главная функция программы
- `fmt.Println()` - функция вывода текста

Запуск программы:
```bash
go run main.go
```

## Базовый синтаксис

### 1. Пакеты
Каждая Go программа состоит из пакетов. Программа начинает выполнение с пакета `main`.

### 2. Импорты
```go
import (
    "fmt"
    "math"
)
```

### 3. Функции
```go
func add(x int, y int) int {
    return x + y
}
```

### 4. Переменные
```go
var name string = "Gopher"
age := 25 // Краткое объявление
```
